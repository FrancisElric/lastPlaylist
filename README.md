# lastPlaylist
## How does it work?

lastPlaylist.sh <user_name> <mix|recomended|library>

## What it does?

1. Downloads playlist created by last.fm from yt
2. Tags it with artist name, title and youtube id

## Needed dependencies

* eyeD3
* ytdl
* jq
